================================================================================
                            HOW TO GET STARTED
================================================================================

OPTION 1: Quick Start (5 minutes)
   $ cd hackweb
   $ docker-compose up -d          # Start database and redis
   $ npm install
   $ npm run dev                   # Frontend on localhost:3000
   $ cd backend && cargo run       # Backend on localhost:8000

OPTION 2: Manual Setup (30 minutes)
   1. Install PostgreSQL
   2. Create database: createdb coin_srec
   3. Copy .env files and configure
   4. Run database migrations
   5. Start backend: cargo run
   6. Start frontend: npm run dev

See QUICK_REFERENCE.md for detailed commands
